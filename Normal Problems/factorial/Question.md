Factorial of a Number

Problem Statement
Given a non-negative integer n, return the factorial of n.

The factorial of a number n is the product of all positive integers less than or equal to n.

For example:- factorial(5) = 5 * 4 * 3 * 2 * 1 = 120
- factorial(0) = 1
Input- A single integer n
Output- Print the factorial of n

Example 1
Input text 5
Output text 120
Explanation 5! = 5 × 4 × 3 × 2 × 1 = 120

Example 2 Input text 0
Output text 1
Explanation By definition, 0! = 1

Constraints- 0 <= n <= 12

Time Complexity- O(n)

Space Complexity
- O(n) for recursive solution
- O(1) for iterative solution
